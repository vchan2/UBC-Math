We can use this property and rules of exponents (from the previous section) to multiply out algebraic expressions involving parentheses. This is sometimes called expanding the expression.

Factors with More than One Term


If the factors have more than one term, then we must distribute multiple times - for instance, once for each term in one factor. Let's begin with the easiest case of \((a+b)(c+d)\). For the sake of clarity, let us think of \((a+b)\) as \(e\), so we have \(e(c+d) = ec+ed\) from distributivity. But this is \((a+b)c+(a+b)d = ac+bc+ad+bd\) by using distributivity again. Now your turn: using this method, prove the equality
$$(a+b)(c+d+e) = ac+ad+ae+bc+bd+be$$
Notice we get a term for each choice of pair of terms, one from each factor. Then in general, if we multiply a factor with \(n\) terms and a factor with \(m\) terms, we should end up with \(nm\) terms (before collecting like terms). This can be a useful trick to check your work! Here are some special cases which are useful to learn:

Special Cases:
